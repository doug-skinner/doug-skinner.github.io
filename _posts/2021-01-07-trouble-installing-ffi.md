title: Trouble installing Ruby-ffi on Macos
---

While not one of the main reasons I started this blog, as I am a Software Engineer I figured it might also be useful to post things that I've discovered in my day-to-day work that may help future me or others.

This morning, while trying to run [Jeyll](https://jekyllrb.com/), the software that powers this website, I ran into the following cryptic error message about ruby-ffi not installing properly.

```bash
An error occurred while installing ffi (1.13.1), and Bundler cannot continue.
Make sure that `gem install ffi -v '1.13.1' --source 'https://rubygems.org/'` succeeds before bundling.
```
