## Logophile

[Logophile](https://www.npmjs.com/package/logophile) is a  basic node.js logging utility that is a low level wrapper around the [winston](https://github.com/winstonjs/winston) package. Logophile's primary purpose is to output logs in a specified JSON format for easy parsing in log search utilities like AWS CloudWatch or Sumo Search. For more information, please click the link above and read the package's NPM page.
